# Custom UI Components — System Design

## Component Specs

### `CustomButton`
Styled button with configurable label text, label colour, and background colour. Uses `RoundedRectangle(cornerRadius: 10)`.

### `CustomTextField`

| Property | Default |
|----------|---------|
| `cornerRadius` | `8` |
| `borderColor` | `.gray` |
| `borderWidth` | `1` |
| `backgroundColor` | `Color(.systemBackground)` |
